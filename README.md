# Web Server with User Login

Простий веб-сервер на Python із автентифікацією користувачів через SQLite базу даних.

## Опис

Застосунок складається з:
- **server.py**: HTTP-сервер із підтримкою CGI для обробки Python-скриптів.
- **setup_db.py**: Скрипт для створення SQLite бази даних та додавання зразкового користувача.
- **login.py**: CGI-скрипт для перевірки логіну та пароля.
- HTML-сторінки: index.html, logon.html тощо.

Паролі хешуються за допомогою bcrypt для безпеки.

## Порядок запуску

1. **Налаштуйте базу даних**:
   ```
   python setup_db.py
   ```
   Це створить файл `login.db` із таблицею `users` та зразковим користувачем (логін: `Dog`, пароль: `12345`).

2. **Запустіть сервер**:
   ```
   python server.py
   ```
   Сервер запуститься на `http://localhost:8080`.

3. **Тестування**:
   - Відкрийте браузер та перейдіть на `http://localhost:8080/logon.html`.
   - Введіть логін `Dog` та пароль `12345`.
   - При успішному вході побачите повідомлення про успіх.

## Додавання нових користувачів

Редагуйте `setup_db.py` для додавання користувачів або створіть окремий скрипт для вставки в БД.

## Вимоги

- Python 3.x
- Встановлені пакети: `bcrypt` (для хешування паролів)

Встановіть залежності: `pip install bcrypt`
